Aims of survival analysis

The point is to fit a model (curve) to survival data and compare the difference between one or survival curves.
We model the effect of one or more explanatory variables on survival like expression of a gene.

Issues specific to survival analysis

  • It is always incomplete in that people are alive at the end of followup and some are just not followed up
  • There is something called right-hand censoring and left-truncation
  • Right hand censoring is when we know that the subject is alive at the end of a study (ie after a certain date)
  • Left hand censoring is when the subject has died but we don't know when (ie it is less than some value)
  • Right hand censoring is far more common

The models for survival analysis incorporate these issues.

Using the survival package in R

The Surv function

The package is simply called 'survival' and is part of the standard library.
You must create a specific survival object in R using the 'Surv' function. The input is a numeric vector.
The second argument to Surv can be a codified representation of censoring = event. 1(TRUE) for survival that is known, 0(FALSE) for right hand censoring.
For left hand censoring ie a threshold of survival we add another vector.

The survfit function

This function fits a nonparametric model to the data. The data can then represented in a Kaplan-Meier plot

The survdiff function

Performs the statistical test for differences in survival

The Cox proportional hazards function. The coxph function

Used when we want to estimate the coefficients of covariates (eg gene expression) in relation to survival
