Protocol for RNA extraction

Materials and methods

You will need

  1. At least least 50,000 cells according to the original NEBnext RNA-seq library prep kit
  2. 500uL Trizol per sample
  3. 4x DNA clean up column per sample (eg QIAquick)
  4. 255uLAMpure XP beads. Bring to room temperature before use
  5. Fresh 80% ethanol for use with the beads
  6. Unique index primers if you intend on multiplexing

    7. mRNA purification

    Use the NEB mRNA magnetic isolation module.
