Materials

  • Standard tissue culture supplies
  • -80 storage
  • Ice
  • Chilled tubes
  • Bright field microscope
  • 96 well U bottom low adherence plates
  • Invasion matrix #3500-096-03. Store this at -80

96 well template

The 96 well template is here: 96 well template
Make sure you save as!

Preparation of a single cell suspension and seeding the cells

  • Refer to the protocol for glioma stem cell culture to generate a single cell suspension: Gliomasphere culture protocol
  • Gulay says that 2,000 cells per 96 well is too much so try out different dilution of cells for the first assay eg 500, 1,000, 2,000.
  • Plate 35uL of cells and and centrifuge at 200g for 3 minutes at room temperature. Use the centrifuge in the dirty area of the Southey laboratory
    Put the balance in position #1 and the plate with cells in #2. The left dial controls which menu you select and the right dial adjusts the setting.
  • After plating cells in 96 well low adherence plate, wait 2 - 3 days for sphere formation to occur.

3D culture invasion

  • Thaw the invasion matrix for 4 hours on ice, or better thaw overnight at 4 C.
    Also remember to chill the centrifuge for when you spin the invasion matrix.
  • Working on ice, add 35uL of the invasion matrix per well to the 96 well plate.
  • Centrifuge plates at 300 x g at 4°C for 5 minutes in a swinging bucket rotor to eliminate bubbles and position spheroids within the Invasion Matrix towards the middle of the well.
  • Transfer plate to a tissue culture incubator set at 37 ̊C for one hour to promote gel formation of the Invasion Matrix.
    Put the balance in position #1 and the plate with cells in #2.
  • After one hour, add 70 μl of warm (37 ̊C) cell culture medium containing chemoattractant and invasion modulating compounds, if applicable. See section VI.3 for reagent preparation.
  • Incubate the plate at 37 °C in a tissue culture incubator for 3 to 6 days, and photograph the spheroid in each well every 24 hours using the 4X objective.

Photography of the cells

  • Use the phase contrast microscope in the main lab.
  • Adjust the lighting and focus to provide the most contrast between the 3D structure and background. Make sure the polariser matches the objective
  • The use of fluorescence microscopy for cells that express fluorescent protein or have a fluorescent label may also improve contrast and subsequent analysis. eg if you have luciferase tagged your cell line
  • Grit and debris may transfer to the bottom of the 96 well plate during handling; wiping the bottom of the wells with a kim wipe prior to photographing may improve clarity.
  • The assay may be conducted longer than 6 days if desired; the endpoint is usually when the structure size begins to exceed the field of analysis or the cells begin to expire.

Analysis of images

ImageJ is free software. It requires a java 6 virtual machine but is cross platform. You can use metamorph too if you have access.

Using the image J script "measureSpheroidArea.ijm"

This macros takes a directory of spheroid images and prompts the user to set manual thresholds and to draw a circle to identify the spheres.
The macros outputs the segmented, processed images in 8 bit format. It also writes the data to an excel spreadsheet.
To use the macros, place the folder "invasionAssay" in /Applications/ImageJ/macros (I don't do PC :/)

  1. Go to imageJ tool bar plugins -> macros -> run. Select measureSpheroidArea.ijm
  2. Select the source directory (folder) containing your images. Only .tif files will be read.
  3. Select the output folder that will contain your data and the processed images. You must select a different folder or the files will be overwritten. You can simply click "New Folder"
  4. The first image will open and then ask you to set the threshold. If you can't find the threshold pane, it is behind the open image. Drag the threshold pane to another location.
  5. Set the threshold that will identify your sphere by sliding the bars around. Click "Set" and then "OK" on the popup panel"
  6. Next move and adjust the elipse that is in the center of the image. Draw it around the invasive outgrowths.
  7. Click "OK" on the panel that has these instructions
  8. Rinse and repeat for all images in the directory.
  9. When all images are analysed, name the output excel file that contains the data.

First calibrate the number of pixels to a know distance (eg 1mm ruler). In the ImageJ macro "measure spheroid area" this is automated.

  1. Open the image called 1mmRuler.tif
  2. Open image. Go to File/Open, and select image.
  3. Select line tool.
  4. Draw a line the length of the object.
  5. Go to Analyze/Measure, and record pixel number.
  6. Calculate the number of pixels in each mm (eg. 600 pixels/mm). I recommend at least 3 measurements and take the mean.
  7. Record result 1316 pixels/mm
  8. Each time you open ImageJ you will need to set the scale. Go Analyze/ Set scale.
  9. For “Distance in pixels” input value the calibration. Tick the "Global" button to save your settings for this session.
  10. For “Know distance” input “1,000”. For “Unit of Length” input “um”.
